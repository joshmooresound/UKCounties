## UKCounties.json

A JSON file that lists all the UK Counties (Northern Ireland to be added soon).

Counties are listed as key:value pairs, organised by country.

### Schema
```
"Country"
  "Abbreviation": "Full Name"
```

### Example
```
{
  "England": {
    "Beds": "Bedfordshire"
  }
}
```
